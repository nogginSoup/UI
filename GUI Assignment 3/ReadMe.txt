This is our GUI Assignment Part 3

Our HTML file is called GameUI, located in the Src folder

 Below our main HUD screen are a line of buttons used to interact with, and switch screens

	Display HP brings up the health bar
	Reduce HP depletes the health bar
	Enter Water brings up the swimming HUD
	Return To Main Screen takes the user back to the main HUD
	Map displays the in-game map
	Mission Screen displays the mission screen
	GameOver displays the game over screen
